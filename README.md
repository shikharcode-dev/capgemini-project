# 📚 Library Management System (Python)

## 📌 Project Description

This is a menu-driven Library Management System built using Python.
It allows users to manage books by adding, issuing, returning, and viewing them.

---

## 🚀 Features

* ➕ Add new books
* 📖 View available books
* 📤 Issue books to students
* 🔁 Return books
* 💰 Fine calculation for late returns

---

## 🧠 Concepts Used

* Python Functions
* Dictionary (for storing book data)
* Lists (basic understanding)
* Conditional Statements
* Loops (while True)
* Modular Programming (multiple files)
* Datetime module (for issue & return tracking)

---

## 📂 Project Structure

```bash
add_book.py
issue_book.py
return_book.py
show_book.py
utils.py
main.py
```

---

## ▶️ How to Run

1. Open terminal
2. Navigate to project folder
3. Run:

```bash
python main.py
```

---

## 🧪 Sample Workflow

1. Add a book
2. Issue the book to a student
3. Return the book
4. System calculates fine (if late)

---

## 🎯 Key Highlight

* Used dictionary to store structured book data
* Implemented fine calculation logic based on delay
* Clean modular code structure
